# Discord-Bot
An easy to setup Discord bot made with Python

# Requirements 
1. Python

# Libraries 
1. discordpy 
2. Pillow 
3. pymongo

# Installation 
```py
# pip install discordpy 
# pip install Pillow
# pip install pymongo 
```

# Setup
1. Open `protected_vars.env`
2. Set `USERNAME` to your `MongoDB` username
